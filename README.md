# React + TypeScript + Vite
This read me tells about the project and shows steps to run it.

Steps to run project:
1. clone repo
2. run `npm i`
3. If installed successfully, run `npm run dev`
4. This will generate a local development url for preview. Open this url in browser.

Libraries used:
1. React Icons
2. Axios
3. React-Data-Table-Component
4. React Responsive
5. React Redux

End Points used:
1. Dummy JSON (get, post requests)
